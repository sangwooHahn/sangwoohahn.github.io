How can the font be personal?

People have their voices and gestures. I studied how these individual differences can be express in the digital world. Text created in a digital environment is designed by a user other. To personalize the font, the user needs to create it themselves. If the design a font by connecting it to something of an individual, it will be a personal font.

Elements from the person

Without changing the font design through commands, the elements of the user and the elements of the font should be directly connected to transform them intuitively.

Voice

It is easy to issue various commands. Variables include tone, pitch, and volume.

Body motion

Many variables such as arm, leg, shoulder and, body position.

Facial expression

Only limited expressions are recognized. There is a variable eye, nose, and mouth.

Hand motion

It's easy to command, but not intuitive to connect to the font.
