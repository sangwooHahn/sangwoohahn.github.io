Thesis / Interaction Design

Transfont

The project started with the question, “Why don’t people make individual fonts?” and “In which way the font can be more reflective to the individual?” To answer this question, the Transfont connects through the interaction between the font and the users.

Interaction

To deliver the connection between the font and the users, “Interaction” is set as the main keyword. The user’s behavior reflects in the font and appears in the appearance of the font. Through this process, users perceive the font that reflects their actions as their font.

UX Research

Transfont is designed through web pages to reach many users. Users can make fonts through their webcam and microphone. An alternative to webcams and microphones is to provide control bars and the text fields. Users can experience real-time fonts design that respond to their behavior.

Resolutions

Transfont is a web-based project that easily accessible through the Internet. Various elements transform into user-specific combinations, allow them to create their unique fonts. The font design by the user can be download as an OTF format. Users can install this file and use the font.
